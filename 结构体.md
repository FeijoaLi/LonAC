### T1. 谁是第一名？

**题目描述**
期末考试结束了，班级里有 $N$ 名学生。每个学生包含两项信息：**姓名**（字符串，无空格）和 **总分**（整数）。
请编写程序找出总分最高的学生，并输出他的姓名。
（如果有多名学生分数一样高，输出输入顺序靠前的那个）。

**输入格式**
第一行一个整数 $N$ ($1 \le N \le 100$)。
接下来 $N$ 行，每行包含一个字符串和一个整数，表示姓名和分数。

**输出格式**
一行，分数最高的学生姓名。

**输入样例**

```
3
ZhangSan 90
LiSi 95
WangWu 88
```

**输出样例**

```
LiSi
```

### T3. 奖学金评定 (多关键字排序 - 重点)

**题目描述**
某校奖学金评定规则如下：

1.  先按 **总分** 从高到低排序。
2.  如果总分相同，按 **语文成绩** 从高到低排序。
3.  如果语文成绩也相同，按 **学号** 从小到大排序（输入顺序即为学号，从 1 开始）。

输入 $N$ 名学生的 **语文**、**数学**、**英语** 三科成绩。请输出排名前 5 的学生的学号和总分。

**输入格式**
第一行一个整数 $N$。
接下来 $N$ 行，每行 3 个整数，依次是语文、数学、英语成绩。
（注意：你需要自己计算总分，并记录学号）。

**输出格式**
共 5 行（如果 $N < 5$ 则输出 $N$ 行），每行两个整数：`学号 总分`。

**输入样例**

```
6
90 90 90
80 100 90
80 85 80
90 95 90
90 90 90
80 100 89
```

**输出样例**

```
4 275
1 270
5 270
2 270
6 269
```

*(解释：1号和5号总分都是270，语文都是90，所以学号小的1号排在前面)*
