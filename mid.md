### T2. 采购铅笔 

**题目描述**
老师需要为班级采购 $n$ 支铅笔。
文具店有 3 种不同包装的铅笔（每种包装的铅笔数量不同，价格也不同）。
例如：

1.  每包 2 支，价格 2 元。
2.  每包 50 支，价格 30 元。
3.  每包 30 支，价格 27 元。
    老师只能选择**其中一种**包装购买，且为了够用，购买的总数量必须 $\ge n$（多买的可以留着）。
    请问老师最少需要花多少钱？

**输入格式**
第一行一个整数 $n$（需要铅笔数量）。
接下来 3 行，每行两个整数 $num, price$，表示该种包装的数量和价格。

**输出格式**
一个整数，表示最少花费。

**输入示例**

```
57
2 2
50 30
30 27
```

**输出示例**

```
54
```

*(解释：买第1种需29包，58元；买第2种需2包，60元；买第3种需2包，54元。54最小)*
*(提示：核心是计算 `ceil(n / num) * price`)*

### T3. 蜗牛爬井

**题目描述**
一只蜗牛在 $H$ 米深的井底。
白天它能向上爬 $U$ 米，但是晚上睡觉时会往下滑 $D$ 米。
请问蜗牛在第几天能爬出井口？（只要白天爬的高度超过或等于 $H$ 就算出来，不会再滑下去）。

**输入格式**
一行三个整数 $H, U, D$。
（保证 $U > D$，否则永远爬不出来）。

**输出格式**
一个整数，表示第几天。

**输入示例**

```
10 3 1
```

**输出示例**

```
5
```

*(解释：
Day1: 爬3(高度3)，滑1(高度2)
Day2: 2+3=5, 5-1=4
Day3: 4+3=7, 7-1=6
Day4: 6+3=9, 9-1=8
Day5: 8+3=11 \>= 10，成功！)*

### T4. 吃苹果

**题目描述**
小明喜欢吃苹果。他现在有 $m$ 个苹果。
他吃苹果的速度非常恒定：每 $t$ 分钟能吃掉一个苹果。
如果你在 $s$ 分钟后来观察他，请问他还剩下几个完整的苹果？
（如果苹果被咬了一口，就不算完整的了。如果时间太长苹果都被吃光了，就输出 0）。

**输入格式**
一行三个整数 $m, t, s$。

**输出格式**
一个整数，表示剩下的完整苹果数量。

**输入示例**

```
50 10 200
```

**输出示例**

```
30
```

*(解释：200分钟吃了 200/10 = 20 个。还剩 50-20=30。注意处理除不尽的情况)*

### T5. 数字根 (Digital Root)

**题目描述**
对于一个正整数 $n$，我们将它的各个数位上的数字相加，得到一个新的数。
如果这个新数是个位数，那么它就是 $n$ 的“数字根”。
如果这个新数还是多位数，我们就重复上述过程，直到它变成个位数。
例如 $n = 356$：

1.  $3+5+6 = 14$
2.  $14$ 是两位数，继续：$1+4 = 5$
3.  $5$ 是个位数，结束。结果为 5。

**输入格式**
一个正整数 $n$。

**输出格式**
一个整数（0-9之间）。

**输入示例**

```
356
```

**输出示例**

```
5
```
