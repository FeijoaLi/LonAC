## T1. 陶陶的超市大采购 (Easy)

### 题目描述

陶陶带了 $M$ 元钱去超市买零食。超市里有 $n$ 袋零食，第 $i$ 袋零食的价格是 $p_i$。
陶陶是个吃货，他希望买到的**零食数量尽可能多**（不关心好不好吃，只关心数量）。
请问他最多能买几袋？

### 输入格式

第一行两个整数 $n, M$。
第二行 $n$ 个整数，表示每袋零食的价格。

### 输出格式

一个整数，表示最多能买几袋。

### 样例输入

```
5 20
10 5 4 8 3
```

### 样例输出

```
4
```

*(解释：买 3, 4, 5, 8，共需 20 元，正好买 4 袋。如果买 10 元的，只能买少)*

## T2. 组装最大的整数 (Easy)

### 题目描述

给你 $n$ 个个位数字（0-9）。请你将这些数字拼接成一个**最大的**多位数。
例如：有数字 `1, 9, 5`，拼接成最大的是 `951`。

### 输入格式

第一行一个整数 $n$。
第二行 $n$ 个整数（0-9之间）。

### 输出格式

一个拼接后的最大整数（以字符串形式输出）。

### 样例输入

```
4
0 5 9 2
```

### 样例输出

```
9520
```

## T3. 勇者斗恶龙 (Easy)

### 题目描述

勇者要挑战 $n$ 只怪物。勇者当前的战斗力是 $A$。
每只怪物的防御力是 $d_i$。
只有当 **勇者战斗力 $A$ \> 怪物防御力 $d_i$** 时，勇者才能战胜它。
请问勇者最多能战胜多少只怪物？

### 输入格式

第一行两个整数 $n, A$。
第二行 $n$ 个整数，表示怪物的防御力。

### 输出格式

一个整数，表示能战胜的怪物数量。

## T4. 贪吃的小猫 (Easy - 分饼干)

### 题目描述

有 $n$ 只小猫，第 $i$ 只小猫的胃口值为 $g_i$。
你有 $m$ 块饼干，第 $j$ 块饼干的大小为 $s_j$。
如果 **饼干大小 $s_j \ge$ 小猫胃口 $g_i$**，这只小猫就能吃饱。每只小猫最多只能吃一块饼干。
请问你最多能让多少只小猫吃饱？

### 输入格式

第一行两个整数 $n, m$。
第二行 $n$ 个整数（猫的胃口）。
第三行 $m$ 个整数（饼干大小）。

### 输出格式

最多能喂饱的小猫数量。

### 样例输入

```
2 3
10 20
5 12 25
```

### 样例输出

```
2
```

*(解释：12给10，25给20)*

## T5. 商店的“买三送一” (Easy+)

### 题目描述

商场举办活动：如果你一次买 3 件或以上商品，其中**最便宜**的那件商品免费。
你想买 $n$ 件商品，价格分别为 $p_1, p_2, \dots, p_n$。
请问买下所有商品最少需要花多少钱？

### 输入格式

第一行一个整数 $n$。
第二行 $n$ 个整数。

### 输出格式

一个整数，表示最小花费。

### 样例输入

```
4
10 20 30 40
```

### 样例输出

```
80
```

## T7. 只有一次机会的股票 (Intro)

### 题目描述

已知未来 $n$ 天的股票价格。
你只能**进行一次**买入和**一次**卖出（先买后卖）。
请问你最多能赚多少钱？（如果所有时间卖出都亏本，则不买，收益为 0）。

### 输入格式

第一行 $n$。
第二行 $n$ 个整数。

### 输出格式

最大利润。

### 样例输入

```
5
7 1 5 3 6
```

### 样例输出

```
5
```

*(在价格 1 时买入，6 时卖出)*

## T8. 翻硬币 (Easy)

### 题目描述

桌上从左往右放置了 $n$ 枚硬币，有的正面朝上（用 1 表示），有的反面朝上（用 0 表示）。
我们的目标是让所有硬币都**正面朝上**（全变 1）。
每次操作，你可以选择**两个相邻**的硬币，同时翻转它们（0变1，1变0）。
请问最少操作几次能达成目标？（题目保证一定有解）。

### 输入格式

两个字符串（初始状态和目标状态，这里简化为目标全是 1）。
输入一行字符串（仅含 '0' 和 '1'）。

### 输出格式

最少操作次数。

### 样例输入

```
0110
```

### 样例输出

```
3
```
