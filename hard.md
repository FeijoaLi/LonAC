### T1. 连接断开的铁链 (基础镜像题)

**题目描述**
工厂里有 $n$ 段长短不一的铁链。现在需要将它们焊接成一条完整的长铁链。
每次焊接可以将两段铁链连接在一起，消耗的时间等于这两段铁链的长度之和。
请你计算出将所有铁链焊接成一条，最少需要消耗多少时间？

**输入格式**
第一行一个整数 $n$。
第二行 $n$ 个整数，表示每段铁链的长度。

**输出格式**
一个整数，表示最小消耗的时间。

**输入示例**

```
4
5 1 2 4
```

**输出示例**

```
22
```

*(解释：先连 1和2 变成3 (消耗3)；现在有 5, 4, 3。再连 3和4 变成7 (消耗7)；现在有 5, 7。最后连 5和7 变成12 (消耗12)。总消耗 3+7+12=22)*

### T3. 史莱姆进化 (游戏化描述)

**题目描述**
游戏里有 $n$ 只初级史莱姆，每只史莱姆有一个“体积值”。
玩家可以使用“融合技能”：每次选择两只史莱姆进行融合，变成一只更大的史莱姆。
新史莱姆的体积 = 两只原史莱姆体积之和。
融合需要消耗魔法值，消耗量 = 新史莱姆的体积。
请问把所有史莱姆融合为 1 只巨大的史莱姆，最多需要消耗多少魔法值？

**输入格式**
第一行 $n$。
第二行 $n$ 个整数，表示史莱姆的体积。

**输出格式**
一个整数，最小魔法消耗。

**输入示例**

```
3
8 5 8
```

**输出示例**

```
37
```
